# Mobile Security Application

A Python-based security system utilizing face recognition for user authentication.

## Details
- **Developer**: [[Syed Abdul Rafay]]
- **Technologies**: Python, OpenCV, Face Recognition API, SQLite
- **Key Features**:
    - Real-time face detection and user authentication.
    - Backend logging module to track recognized users and flag unknown individuals.
    - Computer vision algorithms for face registration and feature extraction.
    - Secure user credential management using SQLite.
