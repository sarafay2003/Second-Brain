# Medical Report Explainer Chatbot

An AI-powered chatbot application designed to simplify complex medical terminology for patients.

## Details
- **Developer**: [[Syed Abdul Rafay]]
- **Technologies**: Python, LangChain, Hugging Face, FAISS, Streamlit, NLP
- **Key Features**:
    - Conversational AI workflows using LangChain.
    - Semantic search over vectorized medical documents via FAISS.
    - Retrieval-Augmented Generation (RAG) for enhanced response accuracy.
    - Interactive web interface built with Streamlit.
